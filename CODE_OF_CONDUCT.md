# Code of Conduct

## Pledge

This project is a learning resource. I welcome questions, corrections, and contributions from everyone, regardless of experience level.

## Expected Behaviour

- Be respectful and constructive
- Focus on the content, not the person
- Accept feedback gracefully
- Help others learn

## Unacceptable Behaviour

- Harassment or discrimination
- Dismissive or condescending responses
- Personal attacks

## Enforcement

Issues can be reported by opening a GitHub issue or contacting the maintainer directly. Violations may result in removal from the project.

This Code of Conduct is adapted from the [Contributor Covenant](https://www.contributor-covenant.org/).
